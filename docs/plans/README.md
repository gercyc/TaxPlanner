# Planos de Execução — TaxPlanner

Esta seção reúne **planos prontos para implementação**. Cada plano descreve
fases, tarefas, critérios de aceite e ordem de execução. Quando a execução
começa, o plano é movido/copiado para `docs/work-plans/` com um log de
execução.

## Como criar um plano

1. Usar o comando `/pwf-plan` ou seguir o esqueleto abaixo.
2. Salvar em `docs/plans/AAAA-MM-DD-<slug-curto>.md`.
3. Numerar e referenciar dependências entre planos.

## Esqueleto de plano

```md
# Plano — Título

- **Data**: AAAA-MM-DD
- **Owner**: ...
- **Status**: Rascunho | Pronto | Em execução | Concluído

## Objetivo
...

## Fases
1. **Fase 1** — ...
2. **Fase 2** — ...

## Critérios de aceite
- ...

## Riscos
- ...

## Referências
- ...
```
