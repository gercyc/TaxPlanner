# Work-Plans — TaxPlanner

Work-plans são **planos em andamento** com log de execução (data/hora,
conquistas, desvios). São derivados de `docs/plans/`.

## Como criar um work-plan

1. Copiar o plano correspondente de `docs/plans/`.
2. Adicionar seção **Execution Log** no final.
3. Atualizar status das tarefas conforme avança.
4. Ao concluir, mover/renomear para indicar conclusão.

## Esqueleto de work-plan

```md
# Work-Plan — Título

- **Data de início**: AAAA-MM-DD
- **Plano original**: `../plans/AAAA-MM-DD-<slug-curto>.md`
- **Status**: Em andamento | Concluído

## Tarefas
- [ ] ...
- [x] ...

## Execution Log
- AAAA-MM-DD — Início da fase 1.
- AAAA-MM-DD — ...
```
